Produce a concise end-of-session debrief. Run `git status` and `git log --oneline -5` first, then fill in:

**Shipped:**
- Commits on main (hashes + one-line summary each)

**Uncommitted:**
- Files still dirty + why

**Half-done:**
- Anything started but not finished + next step to resume

**Next safe stopping point:**
- What to do first next session

**Memory:**
- Anything worth adding to `MEMORY.md` (or "nothing")

Extra note from Fraser: $ARGUMENTS

Keep it under 20 lines total. No preamble.
